## ↓報告↓

犯罪行為の加害者によって悪用されたサイトをユーザーが報告できるプラットフォームのリスト集です。これは、詐欺やデータ漏えいの急増に対する予防策としての役割も担っており、多くの場合、リンク型のウェブサイトとして現れます。善良で責任ある人間として、そのような発見を、当該案件を取り扱う権限を持つ機関へ報告することが大切です（─‿‿─）
